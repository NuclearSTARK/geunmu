v5.4.6 - 홈화면 아이콘 흰 배경 문제

무엇이 바뀌었나요
- icon-180.png / icon-192.png / icon-512.png를 첨부해주신 남색 배경 SP 방패 아이콘으로 교체했습니다.
- 캐시 때문에 안 바뀌어 보이는 걸 막으려고 index.html / manifest.json의 아이콘 경로에
  ?v=5.4.6 버전 표시를 붙였습니다.

⚠️ 꼭 해야 하는 작업 (제일 중요)
아이폰은 "홈화면에 추가"를 하는 "그 순간"의 아이콘 이미지를 캡처해서 저장해버립니다.
그래서 서버 파일만 바꿔서 새로 업로드해도, 이미 홈화면에 추가되어 있는 기존
"Seul Police" 아이콘은 자동으로 안 바뀝니다. 흰 배경 아이콘이 계속 보이는 것도 이 캐시 때문일 가능성이 큽니다.

1. 이 zip 파일 전체를 GitHub 저장소 루트에 덮어쓰기 업로드
2. 홈화면에 있는 기존 "Seul Police" 아이콘을 길게 눌러 삭제
3. Safari에서 사이트 다시 접속 (강력 새로고침 또는 캐시 지우기)
4. 공유 버튼 → "홈 화면에 추가"로 다시 추가

이 순서대로 하면 남색 배경 아이콘으로 정상적으로 보일 거예요.
